== Description ==

The GDPR (General Data Protection Regulation) is a set of instructions for companies that collect and process EU user data on the Internet.
The new regulation is aimed at improving the level of protection and giving EU residents wide control over their data.

### GDPR Compliance & Cookie Consent Plugin Key Features:
* Privacy & Cookie Consent
* GDPR Consent checkboxes
* Cookie Blocker
* Data Access
* Right to be forgotten

### Integrated Plugins:
* Contact Form 7
* Gravity Forms
* Woocommerce
* BuddyPress
* MailChimp
* WordPress Comments

== Installation ==
This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Please find more details on Plugin Installation in documentation
4. Set Up Page in Menu -> GDPR.
